# Hey, I'm Nick 👋

Cybersecurity student finishing up at NJCU (May 2026) with 6 years of building software across frontend, backend, and infrastructure. I split my time between New Jersey and Tbilisi, Georgia, working on multiple products simultaneously — e-commerce platforms, fintech apps, mobile ordering systems — while studying security full-time.

I care about self-hosted infrastructure, privacy-preserving systems, and building tools that solve real problems. Most of my work lives on private repos for clients, but I'm actively building in the open now.

| Project | What it is | Stack |
|---------|-----------|-------|
